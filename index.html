Quick Start

Run locally (Streamlit)

pip install -r requirements.txt
streamlit run app.py

Requires Python and GDAL dependencies on Linux.

Fast actions

  • Download full states/districts GeoJSON from the sidebar.
  • Select regions → Export selection as GeoJSON.
  • Apply buffer (km) → Preview → Download.
  • Draw connection lines and save the map as PNG.
